Bad UX Museum

The World's Most Unusable Museum

Tools Used: WIX, Figma, Canva

Overview

The Bad UX Museum is a brutalist digital shrine to the worst UX crimes found across the wild web — from dropdown disasters to unholy form layouts. This project is not only a hall of shame but also a design redemption arc where each bad interface is paired with a clean, accessible redesign. Built using Wix, the museum is a critique, comedy, and UX masterclass all in one.

Tech Stack

  • Wix Studio
  • Custom Styling via Wix Editor
  • UX Copywriting & Narrative Mapping
  • Figma (for redesign drafts)

UX Process

1
Discover: Finding the Flawed
We started by immersing ourselves in a digital jungle of bad UX — clunky carousels, chaotic forms, and navigation nightmares. This phase was about collecting real-world design disasters and classifying them by type and impact.
2
Define: Naming the Nightmares
After identifying recurring patterns, we categorized the sins into themed "exhibit rooms" like Cognitive Overload, Visual Confusion, and Navigation Nightmares. Each category framed a specific UX issue to focus on.
3
Ideate: Redesign Redemption
Instead of simply mocking bad design, we brainstormed user-friendly alternatives for each UX horror. Accessibility, readability, and logic guided our redesign ideas.
4
Design: From Cringe to Clean
Using Figma, we created side-by-side visual makeovers of each disaster. We maintained brutalist visual humor while ensuring each redesign followed good UX principles.
5
Deliver: Building the Virtual Exhibit
The entire project was brought to life on Wix Studio, turning the designs into an interactive scroll-based museum. Humor, critique, and insight blended together to engage and educate users.

Visual Design

Landing Page

A bold brutalist intro sets the mood — scroll, laugh, scream. But also: learn. The color palette intentionally leans muted and earthy, letting content (and chaos) take center stage.

Cognitive Overload

This room highlights sin #1: mental burden. From endless forms to redundant CAPTCHAs, the pain points are both shown and solved — complete with redesigned layouts.

Visual Confusion

The ultimate style crimes — inconsistent buttons, camouflage CTAs, red confirm buttons. This exhibit lays it bare and proposes a harmonious color-coded fix.

Navigation Nightmare

Everything from disappearing navbars to endless loops of despair. This exhibit is your UX horror story with a roadmap to clarity and coherence.

Outcomes

Reflections

This project was cathartic — a roast and a remedy rolled into one. I loved turning chaos into clarity. My biggest takeaway? Humor and horror can work together to teach something powerful in UX.
