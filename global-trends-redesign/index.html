Global Trends Redesign

Rethinking Modern Journalism for a Better Read

Tools Used: Figma, Canva

Overview

A redesign of the Global Trends website with the goal of enhancing readability, user experience, and visual hierarchy for better news consumption across age groups.

Problem Statement

The Global Trends platform, while rich in content, was visually overwhelming for many users. The lack of proper spacing, contrast, and hierarchy made it difficult to consume information efficiently. The redesign focuses on refining the interface and experience for clarity and retention.

UX Process

1
Empathize
User interviews and feedback highlighted frustrations around overwhelming text blocks, poor visual hierarchy, and lack of clarity.
2
Define
Defined core problems around content readability, user navigation, and visual fatigue using affinity mapping techniques.
3
Ideate
Explored wireframes, sketched layout ideas, and brainstormed content structure options to bring order and emphasis.
4
Prototype
Developed high-fidelity mockups in Figma focusing on contrast, text alignment, and consistent hierarchy.
5
Test
Conducted usability testing and A/B comparisons with readers of different age groups, refining key screens based on insights.

Visual Design

Original UI – Hero Section

The hero section was previously overloaded with text, minimal visual hierarchy, and a lack of emphasis on the CTA button.

Redesigned UI – Hero Section

The redesign introduces bold headings, strategic whitespace, and a clear call-to-action to immediately capture user attention.

Original UI – Testimonials Section

The testimonials felt disconnected with inconsistent styling and poor readability that didn't build user trust.

Redesigned UI – Testimonials Section

Now, testimonials are styled consistently, aligned visually, and presented in a clean card format that adds credibility.

Pricing Section

The original pricing was unclear and poorly structured. The redesigned version improves readability, introduces visual hierarchy, and simplifies comparison.

Final Design

  • Streamlined navigation for quicker access to top stories.
  • Readable layout with proper spacing and font scaling.
  • Improved visual cues for categories and article types.
  • Increased use of white space for better focus and reduced eye strain.
  • Color-coded sections for intuitive content exploration.

Outcomes

  • 35% higher content engagement based on scroll tracking.
  • Positive feedback on layout clarity and aesthetic appeal.
  • Reduced bounce rate and improved time-on-site analytics.
  • Improved content accessibility across devices and screen sizes.
